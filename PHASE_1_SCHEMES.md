# Phase 1: 100-200 Core Government Schemes

## Strategy
Focus on schemes that:
1. Reach millions of citizens
2. Cover all 7-level filtering dimensions
3. Have well-documented eligibility and requirements
4. Are actively disbursing benefits

- ✅ All 7 categories
- ✅ All life stages (Student → Senior)
- ✅ All demographics (SC/ST/Woman/PwD)
- ✅ Urban + Rural regions
- ✅ Multiple income brackets
- ✅ ~150+ million potential beneficiaries

**Deployment Strategy:**
1. Extract 50 Phase 1 schemes using AI extractor
2. Map to 7-level questions
3. Deploy and test
4. Gradually add Phase 2 (50 schemes)

- ✅ Deep, well-documented schemes (not shallow 5062)

- ✅ Time to improve UX/AI as we grow
